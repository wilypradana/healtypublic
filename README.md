## Run Locally

Clone the project

```bash
  git clone https://github.com/wilypradana/healtypublic.git
```

Go to the project directory

```bash
  cd healtypublic
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
